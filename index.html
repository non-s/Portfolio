TakStud

Sistema de Gestão Educacional

SPA educacional com auth real via Supabase e RBAC enforçado no banco por Row Level Security — políticas SQL, não JS. Multi-tenant por escola. Realtime via PostgreSQL LISTEN/NOTIFY: todos os professores veem atualizações sem recarregar. XSS eliminado via esc().

Supabase PostgreSQL RLS RBAC Realtime Multi-tenant
  • Auth email/senha + registro de escola em transação atômica
  • RLS: aluno não deleta nada — o banco rejeita, não o JS
  • CRUD async: alunos, tarefas, comunicados, grade horária
  • Realtime: atualizações ao vivo entre abas da mesma escola

P2P Chat

Chat Peer-to-Peer Real no Browser

WebRTC DataChannel com Supabase Realtime Broadcast como canal de sinalização efêmero. Código de sala de 6 dígitos substitui o fluxo SDP copy-paste. Padrão request-offer resolve race condition de chegada tardia. Após DataChannel abrir, Supabase sai do loop.

WebRTC Supabase Realtime RTCDataChannel ICE · STUN DTLS-SRTP P2P
  • Sinalização automática via Supabase Realtime Broadcast
  • Código de 6 dígitos — sem SDP copy-paste manual
  • Padrão request-offer: funciona mesmo com chegada tardia
  • Canal P2P direto DTLS-SRTP — zero servidor após conexão
